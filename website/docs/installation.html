Installation

RedForge requires Python 3.10+ and is available on PyPI.

Requirements

  • Python 3.10, 3.11, or 3.12
  • pip 23+
  • API keys for your target LLM provider(s)

pip (Recommended)

bash
# Base install (CLI + API server, no provider libs)
pip install redforge

# With specific providers
pip install "redforge[openai]"
pip install "redforge[anthropic]"
pip install "redforge[gemini]"
pip install "redforge[bedrock]"
pip install "redforge[mistral]"

# All providers
pip install "redforge[all-providers]"

# Development / testing
pip install "redforge[dev]"

Source Install

bash
git clone https://github.com/yourorg/redforge
cd redforge
pip install -e ".[all-providers,dev]"

# Verify
redforge --version
redforge list-probes

Docker

bash
docker pull redforge/redforge:latest

# Or build from source
docker build -f docker/Dockerfile -t redforge .

# Run a scan
docker run --rm \
  -e OPENAI_API_KEY="sk-..." \
  -v $(pwd)/results:/results \
  redforge/redforge:latest \
  scan --provider openai --model gpt-4o --authorization research \
       --format html --output /results/report.html

redforge list-probes
