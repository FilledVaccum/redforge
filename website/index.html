47 Attack Probes

Comprehensive coverage: prompt injection, jailbreaks, PII extraction, CBRN, data poisoning, hallucination, supply chain, and 40+ more — all mapped to OWASP LLM Top 10 and MITRE ATLAS.

9 Model Providers

OpenAI, Anthropic, Google Gemini, AWS Bedrock, Azure OpenAI, Mistral, Ollama, and any generic REST endpoint. Add your own adapter in minutes.

🔄

Multi-Turn Attacks

Crescendo escalation, Skeleton Key injection, and PAIR (Prompt Automatic Iterative Refinement). Attacks that succeed where single-turn fails.

17-Strategy Mutation Engine

Automatically mutate payloads with ROT13, Base64, l33t speak, typos, Unicode tricks, academic framing, roleplay, nested context — FuzzyAI-style exhausitve fuzzing.

🛡

Bidirectional Guardrails

InputScanner + OutputScanner pipeline (llm-guard-style). Detect injections before they reach the model, catch credential leaks and malicious code in outputs.

🔍

YARA + Similarity Detection

14 built-in YARA-style rules for pattern matching. Semantic cosine similarity detector catches paraphrased attacks that keyword scanners miss.

📊

Benchmark Evaluation

Standardized Attack Success Rate (ASR) metrics against AdvBench (520), HarmBench (400), and JailbreakBench (100) — comparable across tools.

⚖️

Compliance Mapping

Automated assessment against NIST AI RMF, EU AI Act, ISO 42001, GDPR. Generate compliance gap reports with remediation recommendations.

📄

4 Output Formats

JSON for CI/CD integration, SARIF for GitHub Code Scanning, HTML for stakeholder reports, Markdown for documentation. All sanitized — no raw model output in HTML.

Quick Start

From install to first scan in under 2 minutes.

bash
# Install base package
pip install redforge

# With your provider
pip install "redforge[openai]"         # OpenAI
pip install "redforge[anthropic]"      # Anthropic Claude
pip install "redforge[all-providers]"  # All providers

# Set API key
export OPENAI_API_KEY="sk-..."

# Run first scan
redforge scan --provider openai --model gpt-4o --authorization research
bash
# Pull and run
docker pull redforge/redforge:latest

docker run --rm \
  -e OPENAI_API_KEY="sk-..." \
  -v $(pwd)/results:/results \
  redforge/redforge:latest \
  scan --provider openai --model gpt-4o \
       --authorization research \
       --format html --output /results/report.html

# With docker-compose
docker-compose -f docker/docker-compose.yml up
yaml
# redforge.yaml
provider: openai
model: gpt-4o
authorization: research

# Select probes
probe_categories:
  - owasp_id: LLM01
  - severity: critical

# Compliance frameworks
compliance: [NIST_AI_RMF, EU_AI_ACT]

# Output
output:
  formats: [json, html, sarif]
  directory: ./results
bash
redforge scan --config redforge.yaml
python
import asyncio
from redforge import Scanner

async def main():
    scanner = Scanner(
        provider="openai",
        model="gpt-4o",
        authorization="research",
    )
    report = await scanner.scan()

    print(f"Risk Score: {report.risk_score}/10")
    print(f"Critical: {report.critical_count}")
    print(f"High: {report.high_count}")

    # Save reports
    scanner.save_report(report, "json", "./results")
    scanner.save_report(report, "html", "./results")

asyncio.run(main())

Advanced Attack Modules

Multi-turn orchestration inspired by PyRIT, EasyJailbreak, and cutting-edge research.

Multi-Turn

Crescendo

Gradually escalates requests across 8 turns — starting benign, ending harmful. Each turn builds trust and context.

Multi-Turn

Skeleton Key

Injects false "safe mode" context to convince models their normal safety guidelines are suspended.

Iterative

PAIR Attack

Prompt Automatic Iterative Refinement (Chao et al. 2023). Uses 10 refinement templates to progressively escalate toward goal.

Adversarial

GCG Suffix

Greedy Coordinate Gradient cached suffixes (Zou et al. 2023). 40 payload combinations across 4 base prompts × 10 adversarial suffixes.

Extraction

Repetition Divergence

"Poem poem poem" training data extraction (Nasr et al. 2023). Causes models to regurgitate memorized training data.

Encoding

Typoglycemia / FlipAttack

Scrambles word interiors and reverses characters (LLAMATOR + PyRIT). Bypasses safety filters through obfuscation.
